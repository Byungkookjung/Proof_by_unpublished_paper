Math Model Random Forests

$$ \hat{f(x)} = \sum_{i=1}^{n} c_i \cdot \mathbf{1}(x \in B_i) $$

Where:

  • x is the input feature vector.
  • n is the number of terminal nodes (leaf nodes) in the tree. In our model, we set n = 1000.
  • ci is the prediction value (output) associated with the i-th leaf node.
  • Bi is the disjoint region (box) in the feature space corresponding to the i-th leaf node.
  • 1(x âˆˆ Bi) is the indicator function, which equals 1 if x belongs to Bi, and 0 otherwise.
